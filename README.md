# orangutan-sh

This extension provides basic syntax highlighting for orangutan.

## Building

`vsce package -o ./dist/`

## Installation

`code --install-extension dist/orangutan-syntax-highlighting.vsix`

## Release Notes

### 0.0.3

Added highlighting of:

- `number()`
- `keys()`
- `values()`
- `entries()`

### 0.0.2

- `int()`
